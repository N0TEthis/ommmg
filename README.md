## Скрипты

### Приветствие
**Файл:** `greeting.sh`

Скрипт запрашивает имя пользователя и выводит приветственное сообщение.

```bash
#!/bin/bash

echo "Введите ваше имя:"
read name
echo "Привет, $name! Добро пожаловать!"
```

### Калькулятор суммы
**Файл:** `calculator.sh`

Скрипт запрашивает два числа и выводит их сумму.

```bash
#!/bin/bash

echo "Введите первое число:"
read num1
echo "Введите второе число:"
read num2

sum=$((num1 + num2))
echo "Сумма чисел: $sum"
```

### Проверка четности числа
**Файл:** `even_odd.sh`

Скрипт определяет, является ли введенное число четным или нечетным.

```bash
#!/bin/bash

echo "Введите число:"
read number

if [ $((number % 2)) -eq 0 ]; then
    echo "Число $number четное"
else
    echo "Число $number нечетное"
fi
```

### Создатель структуры проектов
**Файл:** `project_creator.sh`

Создает структуру папок для веб-проекта.

```bash
#!/bin/bash

echo "Введите название проекта:"
read project_name

if [ -z "$project_name" ]; then
    project_name="my-project"
fi

mkdir -p "$project_name"/{css,js,images}

touch "$project_name"/index.html
touch "$project_name"/css/style.css
touch "$project_name"/js/script.js

echo "Структура проекта '$project_name' создана:"
tree "$project_name"
```

### Счетчик строк в файле
**Файл:** `line_counter.sh`

Подсчитывает количество строк в указанном файле.

```bash
#!/bin/bash

echo "Введите путь к файлу:"
read file_path

if [ -f "$file_path" ]; then
    line_count=$(wc -l < "$file_path")
    echo "Количество строк в файле: $line_count"
else
    echo "Файл не найден: $file_path"
fi
```

### Генератор паролей
**Файл:** `password_generator.sh`

Генерирует случайный пароль длиной 8 символов.

```bash
#!/bin/bash

password=$(tr -dc 'A-Za-z0-9!@#$%^&*' < /dev/urandom | head -c 8)
echo "Сгенерированный пароль: $password"
```

### Поиск файлов по расширению
**Файл:** `file_finder.sh`

Ищет файлы по расширению в текущей директории.

```bash
#!/bin/bash

echo "Введите расширение файла (без точки):"
read extension

echo "Найденные файлы с расширением .$extension:"
find . -maxdepth 1 -name "*.$extension" -type f
```

## Инструкция по использованию

1. Сделайте скрипты исполняемыми:
   ```bash
   chmod +x *.sh
   ```

2. Запустите любой скрипт:
   ```bash
   sh script_name.sh
   ```
   или
   ```bash
   ./script_name.sh
   ```

## Требования

- Bash версии 3.2 или выше
- Утилита `tree` (для script4.sh, можно установить через `apt-get install tree` или `brew install tree`)

## Заметки

- Все скрипты протестированы на совместимость с командой `sh script.sh`
- Для корректной работы скрипта `project_creator.sh` рекомендуется установить утилиту `tree`
- Скрипты безопасны для выполнения и не вносят критических изменений в систему
